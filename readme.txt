Nom du projet: 
---------------
Bon Plan Rennes

Auteur et copyright: 
---------------------
Sébastien PHILIPPE pour Rennes Métropole

État du projet: 
----------------
Développement, 1 ère livraison d'avancement.

Rapide description:
--------------------
Pour le client Rennes Métropole:
Créer un portail d’administration pour la gestion des bons plans en mode articles.
Créer un site internet de Bons plans sortie sur Rennes.

Mise ne place, utilisation rapide: 
----------------------------------
Ouvrir un terminal à la racine du projet.
lancer "npm install" pour installer les paquets nodes.js
puis lancer "npm run watch" pour créer les fichiers de distribution.

Dans le fichier Dist crée, double cliquer sur authentification.html et naviguer sur le site à partir de là.
Enjoy :)
