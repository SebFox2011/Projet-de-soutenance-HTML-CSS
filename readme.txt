Rapide description:
--------------------

Mise ne place, utilisation rapide: 
----------------------------------
Ouvrir un terminal à la racine du projet.
lancer npm install pour installer les paquets nodes.js
puis npm run watch pour créer les fichiers de distribution.

Dans le fichier Dist, double cliquer sur authentification.html et naviguer sur le site à partir de la.
Enjoy :)
